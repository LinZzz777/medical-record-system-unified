# 病案借阅管理系统操作说明

## 1. 系统概述

病案借阅管理系统是一个基于Vue 3和Spring Boot的现代化医疗信息管理系统，用于医院内部的病案借阅申请、审批和管理。系统支持普通用户和管理员两种角色，提供完整的借阅流程管理。

### 1.1 技术架构

- **前端**: Vue 3 + Vite + Element Plus
- **后端**: Spring Boot 3.2.0 + MyBatis-Plus
- **数据库**: MySQL 8.0+
- **API文档**: Swagger UI

## 4. 使用流程

### 4.1 用户登录

1. 访问系统首页：[http://localhost:3000/](http://localhost:3002/)
2. 输入用户名和密码
3. 点击登录按钮
4. 登录成功后自动跳转到仪表盘

### 4.2 创建借阅申请

1. 点击左侧菜单"借阅申请"
2. 在病案列表中选择需要借阅的病案（状态为"可借阅"）
3. 点击"发起借阅"按钮
4. 填写借阅信息：
   - 借阅类型（院内借阅/院外借阅）
   - 借阅原因
   - 预计归还日期
5. 点击"提交申请"

### 4.3 审批借阅申请（管理员）

1. 点击左侧菜单"借阅历史"
2. 查看待审批的借阅申请
3. 点击"批准"或"驳回"按钮
4. 驳回时需填写驳回原因

### 4.4 取件操作

1. 点击左侧菜单"借阅历史"
2. 查看已批准的申请
3. 点击"取件"按钮
4. 确认取件操作

### 4.5 归还操作

1. 点击左侧菜单"借阅历史"
2. 查看已取件的申请
3. 点击"归还"按钮
4. 确认归还操作

### 4.6 取消申请

1. 点击左侧菜单"借阅历史"
2. 查看待审批的申请
3. 点击"取消"按钮
4. 确认取消操作

## 6. 系统启动说明

### 6.1 启动后端服务

1. 打开终端，进入后端目录：
   ```bash
   cd d:\cusproject\syst\medical-record-system-unified\backend
   ```
   ""即解压的文件目录下的backed目录，不一定跟我的一模一样""
2. 启动Spring Boot服务：
   ```bash
   mvn spring-boot:run
   ```
3. 等待服务启动完成，控制台显示"Started MedicalRecordSystemApplication"即表示启动成功

### 6.2 启动前端服务

1. 打开新的终端，进入前端目录：
   ```bash
   cd d:\cusproject\syst\medical-record-system-unified\frontend
   ```
   ""即解压的文件目录下的frontend目录，不一定跟我的一模一样""
2. 启动Vue开发服务器：
   ```bash
   npm run dev
   ```
3. 等待服务启动完成，终端会显示访问地址（通常是 [http://localhost:3000/）](http://localhost:3000/）)

## 7. 常见问题

### 6.1 登录失败

- 检查用户名和密码是否正确
- 确认用户状态是否正常

### 6.2 无法创建借阅申请

- 确认病案状态是否为"可借阅"
- 检查是否选择了有效的病案

### 6.3 审批操作失败

- 检查申请状态是否为"待审批"
- 确认是否有操作权限

### 6.4 取件/归还操作失败

- 检查申请状态是否符合要求
- 确认是否为申请本人操作
